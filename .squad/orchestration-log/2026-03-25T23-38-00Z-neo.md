## Technical Summary

Added protocol-aware parsing layer to UdpProxy. Parses RakNet framing and pre-login Minecraft game packets; encrypted post-login packets identified but not decrypted (per Morpheus constraint).

**Packet types handled:**
- UnconnectedPing/Pong (0x01, 0x1c)
- OpenConnectionRequest/Reply (0x05–0x08)
- NewIncomingConnection (0x13)
- DisconnectNotification (0x15)
- Ack/Nack (0xc0, 0xa0)
- DataPacket (0x80–0x8f) with 3-byte LE sequence numbers
- GamePacket (0xfe)
- Unknown packets

**Limitations documented:**
- GamePacket.IsEncrypted always false (tracking login state out of scope)
- No encapsulated-packet parsing inside DataPacket frames
- Offline message ID magic not validated
