## Technical Decisions

1. **Parse-Never-Throw Pattern:** Defensive parsing returns default `ParsedPacket` on malformed input rather than throwing. Caller cannot crash UdpProxy.

2. **GamePacket Encryption State:** Marked `IsEncrypted = false` always. Tracking login handshake state (to determine encryption) is out of scope. Morpheus constraint acknowledged.

3. **No Reliability Layer Parsing:** DataPacket frames contain encapsulated game packets, but parsing frame reliability headers is deferred to Phase 2 of protocol integration.
